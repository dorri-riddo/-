# korean-game

심볼을 이용한 한국어 교육 프로그램


### 특징
1. 보호자가 여러 아이별 학습관리
2. 심볼을 이용하여 각 아이에게 맞춤형 학습판 제작
3. 게임을 통하여 아이가 부담없이 한국어 교육 실시
4. 학습 결과는 보호자가 그래프를 


### 개발환경
1. JSP/Servlet
2. JDBC
3. SQL
